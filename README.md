# NanoNet: Real-Time Polyp Segmentation in Video Capsule Endoscopy and Colonoscopy

This is an unofficial PyTorch implementation of [NanoNet](https://github.com/DebeshJha/NanoNet).


## Requirements

```
pip install -r requirements.txt
```
